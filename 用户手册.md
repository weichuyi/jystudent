# 智慧校园管理平台用户手册

本文档面向系统管理员、教师、学生，说明系统的常用操作流程。

## 一、登录系统

### 访问地址

- 如果运行的是当前主版本 app.py：

```text
http://127.0.0.1:5001
```

### 默认管理员账户

- 用户名：admin
- 密码：weichuy1

### 登录步骤

1. 打开登录页面。
2. 输入用户名和密码。
3. 点击登录。

如果忘记密码：

1. 点击“忘记密码”。
2. 输入学号、工号或用户名提交申请。
3. 联系管理员在操作日志中查看重置 Key。
4. 使用“Key 重置密码”页面，输入账号、Key 和新密码完成重置。

## 二、学生使用说明

### 1. 查看个人信息

登录学生账号后，可在左侧进入“我的信息”查看个人档案。

### 2. 进行选课

操作路径：

- 左侧菜单 -> 课程选择

步骤：

1. 进入课程选择页面。
2. 按学期筛选课程。
3. 点击对应课程后的“选课”按钮。

系统会自动检查：

- 是否已选过该课程
- 课程是否已满员

### 3. 查看已选课程

操作路径：

- 左侧菜单 -> 我的选课

说明：

- 可查看当前已选课程和历史退课状态
- 学生不能自行退课，如需处理请联系管理员

### 4. 查看成绩

操作路径：

- 左侧菜单 -> 成绩管理

学生仅能查看自己的成绩。

## 三、教师使用说明

教师分为两类：

- 任课教师
- 辅导员教师

### 1. 任课教师

任课教师可使用：

- 课程管理：查看自己任课课程
- 成绩管理：录入或修改自己任课课程的成绩

任课教师不能使用：

- 班级管理
- 非本人负责范围的学生管理

### 2. 辅导员教师

如果教师同时是辅导员，则额外可使用：

- 学生信息管理
- 班级管理
- 选课记录管理

### 3. 录入成绩

操作路径：

- 左侧菜单 -> 成绩管理

步骤：

1. 进入成绩管理页面。
2. 找到对应课程和学生。
3. 点击录入成绩。
4. 输入分数并提交。

如果页面中没有目标记录，可使用“手动录入成绩”功能。系统会在必要时自动创建选课记录。

## 四、管理员使用说明

管理员拥有系统全部权限。

### 1. 学生管理

可执行：

- 新增学生
- 编辑学生
- 删除学生
- 批量导入学生

说明：

- 新增学生时会自动创建学生账号
- 删除学生时会同步删除对应用户账号

### 2. 教师管理

可执行：

- 新增教师
- 编辑教师
- 删除教师
- 批量导入教师

说明：

- 新增教师时会自动创建教师账号
- 删除教师时会按关联情况同步删除或禁用账号

### 3. 班级管理

可执行：

- 新增班级
- 编辑班级
- 删除班级
- 设置辅导员

### 4. 课程管理

可执行：

- 新增课程
- 批量导入课程
- 设置主讲教师和协同教师
- 管理课程容量与学期

### 5. 代学生选课

操作路径：

- 左侧菜单 -> 选课记录 -> 代学生选课

步骤：

1. 选择学生。
2. 选择课程。
3. 点击确认选课。

### 6. 成绩管理

管理员可查看和录入所有成绩。

### 7. 用户管理

可执行：

- 新增用户
- 编辑用户
- 删除用户

注意：

- 不能删除自己
- 系统至少保留一个管理员账户

### 8. 操作日志与重置 Key

操作路径：

- 左侧菜单 -> 操作日志

可查看：

- 用户操作记录
- 忘记密码后生成的重置 Key

### 9. 数据导出

操作路径：

- 管理中心 -> 数据导出

可导出：

- 学生数据
- 成绩数据
- 课程数据

### 10. 数据备份与恢复

操作路径：

- 管理中心 -> 备份管理

可执行：

- 创建备份
- 恢复备份
- 下载备份
- 删除备份

### 11. 系统信息与恢复出厂

操作路径：

- 管理中心 -> 系统信息

可查看：

- Python 版本
- 操作系统
- 处理器信息
- 备份目录占用
- 日志目录占用

可执行危险操作：

- 跑路

说明：

- 点击后会弹出确认弹窗
- 需要等待 5 秒后才能点击“确定跑路”
- 执行后会删除所有业务数据，仅保留默认管理员账户

## 五、常见问题

### 1. 学生为什么看不到退课按钮？

因为系统已禁止学生自行退课。若需退课，请联系管理员处理。

### 2. 教师为什么看不到班级管理？

只有辅导员教师才可以管理班级。普通任课教师只能按任课课程管理成绩。

### 3. 为什么管理员和学生的选课入口不同？

- 学生通过“课程选择”进行自助选课
- 管理员通过“选课记录”中的“代学生选课”进行操作

### 4. 如果误点了“跑路”怎么办？

该操作不可恢复。建议执行前先在备份管理中创建数据库备份。
