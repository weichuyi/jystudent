# 智慧校园管理平台 - 快速启动指南

## 📋 项目概述

这是一个企业级智慧校园管理平台，集成了用户认证、权限管理、学生信息、课程安排、选课、成绩管理等功能。支持三种用户角色（管理员、教师、学生），可直接部署或打包为 Windows 可执行程序。

## 🚀 快速开始（3 分钟）

### Windows 用户
1. 进入项目目录
2. 运行 `run.py` 启动应用
   ```
   python run.py
   ```
3. 浏览器自动打开，访问 http://127.0.0.1:5000
4. 默认账号：**admin** / **weichuy1**

### Linux/Mac 用户
```bash
# 安装依赖
pip install -r requirements.txt

# 运行应用
python run.py
```

## 🔧 常见操作

### 创建新用户
1. 登录管理员账户
2. 点击 系统 → 用户管理
3. 实际操作：点击"新增用户"，填写信息并选择角色

### 添加学生
1. 登录管理员或教师账户
2. 点击 管理 → 学生管理
3. 点击"新增学生"，填写基本信息

### 创建课程
1. 管理员 / 教师登录
2. 点击 管理 → 课程管理
3. 点击"新增课程"，设置学分、容量等参数

### 学生选课
1. 学生登录
2. 点击 选课
3. 选择要学的课程，点击"选课"提交

### 录入成绩
1. 教师 / 管理员登录
2. 点击 管理 → 成绩管理
3. 找到对应学生和课程，点击"录入"或"修改"

## 🖥️ 打包为 Windows 可执行程序

若需要在没有 Python 的电脑上运行，可打包为 `.exe` 文件：

```bash
# 1. 安装 PyInstaller
pip install pyinstaller

# 2. 运行打包脚本
python build_exe.py

# 3. 生成的 exe 文件在 dist/ 目录下
# 直接双击 StudentManagementSystem.exe 即可运行
```

### 一键生成测试发布包（推荐）

```bash
# 自动构建 exe + 打包当前测试数据库 + 生成 zip
python package_release.py

# 使用现有 exe 直接打包（更快）
python package_release.py --skip-build

# 不带数据库，只发空库程序
python package_release.py --skip-build --without-db
```

执行后会在 `release/` 下生成测试压缩包，直接发给测试同学即可。

## 🗄️ 数据库架构

系统使用 SQLite 数据库，包含以下主要表：

- **users** - 用户账户（管理员、教师、学生）
- **students** - 学生信息
- **teachers** - 教师信息
- **classes** - 班级信息
- **courses** - 课程信息
- **enrollments** - 选课记录
- **scores** - 成绩记录
- **student_status_changes** - 学籍异动记录
- **operation_logs** - 操作日志

## 🔐 安全特性

- 密码使用 Werkzeug 进行哈希加密
- 全局会话管理，防止未授权访问
- 基于角色的权限控制（RBAC）
- 所有操作都有日志记录

## 📞 故障排除

### 问题：数据库无法初始化
**解决**：删除 students.db 文件，重新运行应用会自动创建

### 问题：登录失败
**解决**：检查用户名密码是否正确，默认管理员账号为 admin / weichuy1

### 问题：无法启动服务器（端口被占用）
**解决**：修改 app.py 中的端口号（默认 5000），或关闭占用端口的程序
